# Shelly Light Control Project

## Overview
Automated light control system using Shelly devices on Raspberry Pi.

## Setup
1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Update `config.yaml` with your Shelly device IPs and preferences

## Running
```bash
python3 light_controller.py
```
